# ИИ Агент - Анализ вакансий на hh.ru на n8n

Разработала еженедельный автоматизированный мониторинг вакансий Data Scientist (Москва) через n8n с AI-анализом навыков и отправкой дайджеста в Telegram.

## Ключевые компоненты workflow:

**Schedule Trigger** — запуск по cron каждую неделю в четверг 09:35.

**HTTP Request** — GET /vacancies API hh.ru (text=Data Scientist, area=1, per_page=10, User-Agent: n8n-career-bot).

**Code Node (JS)** — извлекает description из snippet.requirement + snippet.responsibility, лимит 10 вакансий.

**SplitInBatches** — батчевая обработка через Loop Over Items.

**AI Agent (GigaChat-2-Max)** — извлекает навыки из описания: "список через запятую, без пояснений".

**Aggregate** — собирает все навыки в массив $json.skills.

**AI Agent1** — нормализует навыки, считает частоту, возвращает "навык: количество" (топ-15).

**Code Node1 (JS)** — парсит вывод LLM, форматирует дайджест: "На этой неделе я спарсил 10 свежих вакансий...".

**Telegram Node** — отправка в чат.

Логика: Периодический парсинг → AI-экстракция навыков → агрегация → статистический топ → готовый текст.
